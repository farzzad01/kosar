مستندات پروژه: سامانه نوبت‌دهی ساده با Django و Tailwind CSS
هدف پروژه
ساخت یک وب‌اپلیکیشن ساده برای ثبت نوبت که شامل:

یک صفحه عمومی با فرم ساده (نام، شماره تماس، کار مورد نظر)
ذخیره نوبت‌ها در دیتابیس
پنل مدیریت Django (با یوزر/پسورد) برای مشاهده، جستجو، ویرایش و حذف نوبت‌ها
ظاهر زیبا و responsive با Tailwind CSS (از CDN)
دیپلوی رایگان روی Render.com

ویژگی‌ها

فرم عمومی برای کاربران عادی (بدون لاگین)
نمایش پیام موفقیت پس از ثبت نوبت
پنل ادمین استاندارد Django در آدرس /admin/
مرتب‌سازی نوبت‌ها بر اساس زمان ثبت
کاملاً فارسی و راست‌چین (RTL)

۳. مدل (appointments/models.py)
Pythonfrom django.db import models

class Appointment(models.Model):
    name = models.CharField(max_length=100, verbose_name="نام و نام خانوادگی")
    phone = models.CharField(max_length=20, verbose_name="شماره تماس")
    task = models.TextField(verbose_name="کار مورد نظر")
    created_at = models.DateTimeField(auto_now_add=True, verbose_name="زمان ثبت")

    class Meta:
        ordering = ['created_at']
        verbose_name = "نوبت"
        verbose_name_plural = "نوبت‌ها"

    def __str__(self):
        return f"{self.name} - {self.phone}"

۶. ویو (appointments/views.py)
Pythonfrom django.shortcuts import render, redirect
from django.contrib import messages
from .forms import AppointmentForm

def home(request):
    if request.method == 'POST':
        form = AppointmentForm(request.POST)
        if form.is_valid():
            form.save()
            messages.success(request, 'نوبت شما با موفقیت ثبت شد!')
            return redirect('home')
    else:
        form = AppointmentForm()
    
    return render(request, 'home.html', {'form': form})

۱۱. دیپلوی روی Render.com

تمام کد را به یک ریپازیتوری GitHub push کنید.
در Render.com → New → Web Service → ریپازیتوری را متصل کنید.
تنظیمات:
Build Command: pip install -r requirements.txt && python manage.py migrate
Start Command: gunicorn nobatdehi.wsgi:application

یک دیتابیس PostgreSQL رایگان اضافه کنید (Render خودش DATABASE_URL می‌دهد).
در تنظیمات محیطی (Environment Variables):
PYTHON_VERSION: 3.11 (یا جدیدتر)
DEBUG: False

Deploy کنید.

پس از دیپلوی، با یوزر/پسورد سوپریوزر به /admin/ وارد شوید و نوبت‌ها را مدیریت کنید.
